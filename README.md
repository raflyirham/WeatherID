
# WeatherID

WeatherID is a web application that enables anyone to see the current weather status in any cities in Indonesia. The data is updated every minute using an API from RapidAPI.


## Live Preview
https://weatherid.vercel.app/
## Run Locally

Clone the project

```bash
  git clone https://github.com/raflyirham/WeatherID.git
```

Go to the project directory

```bash
  cd WeatherID
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
